Introduction

This document serves as a User Guide for the Universal Storage Web Tool. This application is designed to be intuitive, and its use does not require extensive technical knowledge. However, this guide serves to inform users about all the features and options available within the tool.

The primary goal of the Universal Storage Web Tool is to provide a single, unified interface to manage files across multiple cloud storage providers, allowing you to overcome individual storage limits and simplify your digital life.

Registration and Authentication

Access to the Universal Storage Web Tool platform is restricted to registered users. The application provides two distinct pages for access control:

  • Login Page: If you already have an account, you can log in directly using your email address and password.
  • Register Page: If you are a new user, you can click the link to the sign up page. An account can be created by providing a valid email address and choosing a password.

After registering, you will be able to log in to the platform with your new credentials and will be automatically redirected to the main Dashboard.

Dashboard

Once you log in, you will land on the Dashboard. This page is the central hub for all activities within the application. From here, you can access every feature without needing to navigate to different pages.

The Dashboard is organized into several key sections:

  • Cloud Connections: This area allows you to connect to or disconnect from your supported cloud services (Google Drive, OneDrive, Dropbox).
  • Storage Statistics: Here, you can see a clear overview of your storage usage. It displays how much space is used versus the total available for each individual cloud, as well as an aggregated total of all your connected accounts.
  • File Management Area: This is the main part of the page, where you can see all the files and folders you have uploaded through the tool. You can initiate uploads, downloads, and deletions directly from this interface.
  • Logout: A clearly visible button allows you to securely log out of your account.

The storage visualization includes:

  • Donut Chart: Shows your aggregate storage usage across all connected providers
  • Provider Bars: Individual progress bars for each connected service (Google Drive, OneDrive, Dropbox)
  • Detailed Stats: Exact byte counts and file quantities for each provider

Uploading

The upload process preserves your complete folder structures:

  • Nested folders are recreated exactly in your cloud storage
  • Relative file paths are maintained
  • Special characters in file/folder names are properly handled

During upload you can:

  • Select multiple files from different folders
  • Upload entire folder trees with their contents
  • Choose which cloud provider should store the files

File Storage and Distribution

The Universal Storage Web Tool maintains your complete files on each connected cloud service while preserving their original folder structures. When you upload a file or folder:

  • The entire file is stored intact on your selected cloud provider
  • Folder hierarchies are exactly replicated in the cloud storage
  • Metadata is stored in our database to track file locations

This approach provides several advantages:

  • Your files remain fully accessible directly through each cloud provider
  • No proprietary file formats or transformations are used
  • Folder structures are maintained exactly as you organized them

Downloading

The download process is the seamless reverse of uploading. When you request to download a file or folder from the Dashboard, you are not just downloading a simple file.

Instead, you send a request to the tool's server, which then automatically locates all the necessary segments from your connected clouds. It retrieves them, verifies their integrity, and reassembles them in the correct order to perfectly reconstruct the original file. The complete, original file is then delivered to your computer for you to save.

File Management

The application provides a clean and familiar file and folder structure. It's important to understand that the tool doesn't store your files directly on its servers.

Instead, it maintains a lightweight and efficient list of your "virtual" files and folders. This list contains only metadata: the file names, sizes, and a map of where each segment is located across your clouds. This approach makes managing your file structure incredibly fast and ensures your data remains on your own cloud accounts.

Connecting Your Cloud Accounts

Currently, the Universal Storage Web Tool allows you to connect to three different cloud services: Google Drive, Microsoft OneDrive, and Dropbox.

The connection with these external applications is created using the OAuth 2.0 protocol, which is the industry standard for secure authorization. When you first connect a drive, you will be redirected to that service's official login page to grant permission. This means we never see or store your cloud account passwords. The service provides our application with a secure access token, which you can revoke at any time from the Dashboard.

Conclusion

In conclusion, the Universal Storage Web Tool is a powerful utility for anyone looking to make the most of their available cloud storage. It helps you overcome storage limits by combining accounts and provides a simple, secure interface to manage all your distributed files as if they were in one place.
